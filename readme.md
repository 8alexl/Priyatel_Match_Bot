# Priyatel Match Bot

Этот Telegram-бот предназначен для поиска и общения пользователей на основе их интересов. Пользователи могут регистрироваться, указывая свою информацию и интересы, а затем находить собеседников с похожими увлечениями и начинать переписку.

## Функции 📋

- **Регистрация пользователей** — возможность зарегистрироваться и указать свои интересы.
- **Поиск собеседников** — находите других пользователей с похожими интересами.
- **Чат** — начинайте чат с выбранным пользователем.
- **Прерывание чатов** — возможность остановить переписку с помощью команды.
- **Интерактивные клавиатуры** — простое и удобное взаимодействие через inline-кнопки и клавиатуры.

## Начало работы 🚀

### Установка

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/telegram-matchmaking-bot.git
   cd telegram-matchmaking-bot
   ```

2. Установите необходимые зависимости:
   ```bash
   pip install pyTelegramBotAPI python-dotenv
   ```

3. Создайте базу данных (если она ещё не создана):
   ```bash
   touch users.db
   ```

4. Получите токен для бота от [BotFather](https://t.me/BotFather). Добавьте его в файл `.env`:
   ```plaintext
   TELEGRAM_BOT_TOKEN='YOUR_BOT_TOKEN_HERE'
   ```

### Запуск

Запустите бота с помощью Python:
```bash
python bot.py
```

### Использование

- Начните общение с ботом с помощью команды `/start`.
- Пройдите регистрацию с помощью `/register` и укажите свои интересы.
- Найдите собеседников с помощью команды `/match` или кнопки `Поиск🔍`.
- Начните чат с выбранным пользователем.
- Прервите чат с помощью команды `/stop`.

## Команды 📖

| Команда       | Описание                                             |
|---------------|------------------------------------------------------|
| `/start`      | Начало общения с ботом                               |
| `/register`   | Регистрация пользователя                             |
| `/match`      | Поиск собеседника с похожими интересами              |
| `/stop`       | Прекращение текущего чата                            |
| `/help`       | Информация о командах                                |

## Примеры использования

1. **Регистрация**: Запустите `/register` и следуйте инструкциям для заполнения анкеты.
2. **Поиск собеседника**: Используйте `/match` или кнопку `Поиск🔍` для нахождения пользователей с похожими интересами.
3. **Начало чата**: Нажмите на кнопку для начала чата с найденным пользователем.
4. **Остановка чата**: Введите `/stop` для завершения текущей беседы.

## Требования

- Python 3.6+
- Библиотеки:
  - [pyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI)
  - [python-dotenv](https://pypi.org/project/python-dotenv/)

## Структура базы данных 🗄️

| Поле       | Тип      | Описание                            |
|------------|----------|------------------------------------|
| `id`       | INTEGER  | Уникальный идентификатор пользователя (chat_id) |
| `nick`     | TEXT     | Никнейм пользователя               |
| `name`     | TEXT     | Имя пользователя                   |
| `age`      | INTEGER  | Возраст пользователя               |
| `sex`      | TEXT     | Пол пользователя                   |
| `interest1`| TEXT     | Интерес пользователя 1             |
| `interest2`| TEXT     | Интерес пользователя 2             |
| `interest3`| TEXT     | Интерес пользователя 3             |
