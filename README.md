## Telegram чат-бот на Python для поиска людей по интересам

Этот проект представляет собой телеграм-бота, который помогает пользователям находить собеседников на основе их интересов. Бот предлагает пользователям пройти регистрацию, указав свои данные и интересы, после чего позволяет искать и общаться с подходящими собеседниками.

## Функции

- `/start` — начать работу с ботом.
- `/register` — начать регистрацию (указать имя, возраст, пол и интересы).
- `/match` — найти собеседников по интересам.
- `/stop` — остановить текущую переписку.
- `/help` — получить помощь по командам бота.
- `/about` — информация о боте.

## Установка и запуск

Для использования бота необходимо выполнить следующие шаги:

### Требования

- Python 3.7 или выше.
- Установленные библиотеки:
  - `telebot`
  - `python-dotenv`

### Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/8alexl/Priyatel_Match_Bot.git
```

2. Перейдите в директорию проекта:

```bash
cd Priyatel_Match_Bot
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Создайте файл `.env` в корне проекта и добавьте токен вашего бота:

```bash
BOT_TOKEN=Your_Token
```

### Запуск

После установки зависимостей и настройки `.env`, запустите бота с помощью следующей команды:

```bash
python bot.py
```
